Python struct 用法

主要函数

  • pack(fmt, v1, v2, …)
    根据所给的fmt描述的格式将值v1，v2，…转换为一个字符串
  • unpack(fmt, bytes)
    根据所给的fmt描述的格式将bytes反向解析出来，返回一个元组
  • calcsize(fmt)
    根据所给的fmt描述的格式返回该结构的大小

主要定义格式

  • i – int
  • f – float
  • d double
  • s string
  • 其他可看官方文档
